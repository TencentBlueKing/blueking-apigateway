# 功能: 统计报表 - 运行数据统计

**模块**: 统计报表
**页面**: /gateways/:id/statistics
**优先级**: P1
**前置条件**: 已登录，且网关存在运行数据

## 场景: 查看统计图表

- **假设** 用户已登录并进入统计报表页面
- **当** 用户选择环境和资源
- **那么** 页面应展示对应条件下的统计图表
- **并且** 支持通过资源选择框输入关键字搜索资源
- **并且** 支持通过蓝鲸应用ID搜索对应的统计数据
- **并且** 支持按状态码（如200）筛选统计数据
- **并且** 支持点击"刷新"按钮更新统计数据

## 场景: 时间范围切换

- **假设** 用户已进入统计报表页面
- **当** 用户点击时间选择框，选择开始时间和结束时间
- **并且** 点击"确定"按钮
- **那么** 页面应展示对应时间范围内的统计报表
- **并且** 支持快捷选择"今天"、"近7天"、"近15天"、"近30天"
- **并且** 支持点击"清除"按钮清除已选择的时间
