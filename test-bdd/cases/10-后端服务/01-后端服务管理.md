# 功能: 后端服务 - 后端服务管理

**模块**: 后端服务
**页面**: /gateways/:id/backend-services
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关

## 场景: 创建后端服务

- **假设** 用户已进入后端服务页面
- **当** 用户点击新建，输入服务名称（必填）、选填描述
- **并且** 选择负载均衡类型，输入后端服务地址和超时时间
- **并且** 点击确定
- **那么** 后端服务新建成功，服务列表中展示新建的后端服务

## 场景: 编辑后端服务

- **假设** 用户已进入后端服务页面，已存在后端服务
- **当** 用户选择后端服务点击编辑
- **并且** 修改基础信息或各环境的服务配置
- **并且** 点击确定
- **那么** 页面提示保存成功，正在发布到对应的环境

## 场景: 删除后端服务

- **假设** 用户已进入后端服务页面，后端服务未关联资源
- **当** 用户选择后端服务点击删除
- **并且** 确认删除操作
- **那么** 后端服务删除成功，服务列表中不再展示该服务

## 场景: 查看服务列表

- **假设** 用户已进入后端服务页面，已存在多个后端服务
- **当** 用户在搜索框中输入服务名称进行搜索
- **那么** 页面展示匹配的后端服务
- **当** 用户输入不存在的服务名称进行搜索
- **那么** 页面展示空结果
