# 功能: 操作记录 - 操作审计记录

**模块**: 操作记录
**页面**: /gateways/:id/audit
**优先级**: P1
**前置条件**: 已登录，且网关存在操作记录

## 场景: 查看操作记录

- **假设** 用户已登录并进入操作记录页面
- **当** 用户点击时间选择框，分别选择开始时间和结束时间
- **并且** 点击"确定"按钮
- **那么** 页面应展示对应时间范围内的操作记录
- **并且** 支持快捷选择"最近5分钟"、"最近1小时"、"最近6小时"、"最近12小时"、"最近1天"、"最近7天"
- **并且** 支持点击"清除"按钮清除已选择的时间

## 场景: 记录筛选

- **假设** 用户已进入操作记录页面
- **当** 用户点击搜索框选择条件"操作对象"进行查询
- **那么** 列表应展示匹配操作对象（如网关）的操作记录
- **并且** 支持通过操作对象、操作类型、操作状态等条件组合搜索
- **并且** 支持通过列表头部筛选按钮按操作对象、操作类型、操作状态筛选
- **并且** 支持自定义表格列设置
