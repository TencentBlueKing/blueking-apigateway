# 功能: 环境变量管理 - 环境变量管理

**模块**: 环境变量管理
**页面**: /gateways/:id/stages/:stage/variables
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在环境

## 场景: 创建环境变量

- **假设** 用户已进入环境概览页面，切换到详情模式并选择环境的变量管理
- **当** 用户点击变量列表旁的编辑（铅笔图标）
- **并且** 输入变量名（由字母、数字、下划线组成，首字符必须是字母，长度小于50个字符）
- **并且** 输入变量值，点击保存并确认修改
- **那么** 页面提示更新成功，变量列表中展示新创建的变量

## 场景: 编辑环境变量

- **假设** 用户已进入环境变量管理页面，已存在变量
- **当** 用户点击变量列表旁的编辑（铅笔图标）
- **并且** 修改变量名或变量值
- **并且** 点击保存并确认修改
- **那么** 页面提示更新成功，变量信息已更新
