# 功能: 资源配置 - 编辑资源

**模块**: 资源配置
**页面**: /gateways/:id/resources
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在可编辑的资源

## 场景: 编辑基础信息

- **假设** 用户已进入资源配置页面，已存在至少一个资源
- **当** 用户点击资源列表中的编辑按钮
- **并且** 修改资源名称、描述、标签等基础信息
- **并且** 点击提交
- **那么** 页面提示更新成功，并提示需生成新版本并发布到指定环境才能生效

## 场景: 编辑后端配置

- **假设** 用户已进入资源编辑页面
- **当** 用户修改后端服务选择、请求方法或请求路径
- **并且** 校验后端地址成功
- **并且** 点击提交
- **那么** 页面提示更新成功
