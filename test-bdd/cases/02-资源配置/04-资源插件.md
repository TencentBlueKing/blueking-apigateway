# 功能: 资源配置 - 资源插件

**模块**: 资源配置
**页面**: /gateways/:id/resources
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在资源

## 场景: 添加插件

- **假设** 用户已进入资源配置页面，已存在至少一个资源
- **当** 用户点击资源插件数对应的数值，点击添加插件
- **并且** 选择插件类型（如跨域资源共享CORS插件），点击下一步
- **并且** 填写插件配置信息（如allow_origins等必填项），点击确定
- **那么** 插件添加成功，资源插件数量增加

## 场景: 编辑插件

- **假设** 用户已进入资源配置页面，资源已绑定插件
- **当** 用户点击资源插件数对应的数值
- **并且** 鼠标悬浮已有插件，点击编辑（铅笔图标）
- **并且** 修改插件配置信息，点击确定
- **那么** 页面提示修改成功

## 场景: 删除插件

- **假设** 用户已进入资源配置页面，资源已绑定插件
- **当** 用户点击资源插件数对应的数值
- **并且** 鼠标悬浮已有插件，点击删除（垃圾桶图标）
- **并且** 确认停用
- **那么** 插件停用成功，资源插件数量减少
