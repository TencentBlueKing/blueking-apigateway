# 功能: 资源配置 - 创建资源

**模块**: 资源配置
**页面**: /gateways/:id/resources
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关且后端服务地址不为空

## 场景: 创建资源并配置后端

- **假设** 用户已进入资源配置页面，后端服务地址已配置
- **当** 用户点击新建，输入资源名称（必填）、选填描述和标签
- **并且** 配置请求方法和请求路径（必填）
- **并且** 选择后端服务，配置后端请求方法和请求路径，校验地址成功
- **并且** 点击提交
- **那么** 资源创建成功，资源列表中展示新创建的资源

## 场景: 名称校验

- **假设** 用户已进入新建资源页面
- **当** 用户未填写资源名称并点击提交
- **那么** 页面提示"请填写名称"
- **当** 用户输入的资源名称不是由字母、数字、下划线（_）组成
- **那么** 页面提示"由字母、数字、下划线（_）组成，首字符必须是字母，长度小于256个字符"

## 场景: 请求配置

- **假设** 用户已进入新建资源页面
- **当** 用户选择请求方法、输入请求路径，并启用WebSocket
- **并且** 配置请求参数（支持通过JSON生成请求参数、新增字段、删除字段）
- **并且** 配置响应参数（新增状态码，支持修改删除状态码、通过JSON生成）
- **那么** 请求配置和响应参数保存成功
