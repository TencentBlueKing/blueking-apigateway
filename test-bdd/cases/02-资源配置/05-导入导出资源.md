# 功能: 资源配置 - 导入导出资源

**模块**: 资源配置
**页面**: /gateways/:id/resources
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关

## 场景: 导入资源

- **假设** 用户已进入资源配置页面
- **当** 用户点击导入，选择资源文档
- **并且** 选择文档语言（中文或英文），上传Swagger文件
- **并且** 点击下一步，确认导入
- **那么** 资源导入成功，资源列表中展示导入的资源

## 场景: 导出资源

- **假设** 用户已进入资源配置页面，已存在资源
- **当** 用户点击导出，选择全部资源
- **并且** 选择导出格式为YAML格式或JSON格式
- **并且** 点击确定
- **那么** 资源导出成功，导出数据与全部资源数据保持一致
- **当** 用户勾选部分资源，通过批量操作导出
- **并且** 选择导出格式，点击确定
- **那么** 仅导出所选资源，导出成功
