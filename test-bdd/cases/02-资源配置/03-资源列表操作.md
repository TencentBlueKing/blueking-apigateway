# 功能: 资源配置 - 资源列表操作

**模块**: 资源配置
**页面**: /gateways/:id/resources
**优先级**: P2
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在多个资源

## 场景: 搜索资源

- **假设** 用户已进入资源配置页面，已存在多个资源
- **当** 用户在搜索框中输入资源名称进行搜索
- **那么** 页面展示匹配的资源
- **当** 用户选择搜索条件为前端请求路径并输入路径进行搜索
- **那么** 页面展示对应前端请求路径的资源
- **当** 用户选择搜索条件为前端请求方法并输入方法进行搜索
- **那么** 页面展示对应请求方法的资源

## 场景: 标签筛选

- **假设** 用户已进入资源配置页面，资源已配置标签
- **当** 用户通过标签筛选资源
- **那么** 页面仅展示包含所选标签的资源
- **当** 用户在新建资源时创建、编辑或删除标签
- **那么** 标签操作成功，可正常勾选使用

## 场景: 批量操作

- **假设** 用户已进入资源配置页面，已存在多个资源
- **当** 用户勾选多个资源，点击批量操作中的编辑资源
- **那么** 可批量编辑所选资源的属性
- **当** 用户勾选多个资源，点击批量操作中的删除资源并确认
- **那么** 所选资源删除成功
- **当** 用户未勾选资源直接点击批量操作中的编辑资源
- **那么** 页面提示"请先勾选资源"
