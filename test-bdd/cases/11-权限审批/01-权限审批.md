# 功能: 权限审批 - 权限审批管理

**模块**: 权限审批
**页面**: /gateways/:id/permissions/approvals
**优先级**: P1
**前置条件**: 已登录，且存在待审批的权限申请单据

## 场景: 查看审批列表

- **假设** 用户已登录并进入权限审批页面
- **当** 用户点击"权限管理"-"权限审批"菜单
- **那么** 页面应展示待审批的权限申请列表
- **并且** 列表中应显示蓝鲸应用ID、申请人、申请资源等信息
- **并且** 支持通过蓝鲸应用ID、申请人、资源名称等条件搜索审批单据
- **并且** 支持切换授权维度（按资源/按网关）进行筛选
- **并且** 支持通过左下角切换每页显示条数（10/20/50/100）

## 场景: 审批通过

- **假设** 审批列表中存在待审批的权限申请单据
- **当** 用户选择需要审批的单据，点击"全部通过"
- **并且** 选填备注信息后点击"确定"
- **那么** 页面应提示"操作成功"
- **并且** 支持批量选择多条单据后点击"批量审批"进行全部通过
- **并且** 审批通过后该单据从待审批列表中移除

## 场景: 审批驳回

- **假设** 审批列表中存在待审批的权限申请单据
- **当** 用户选择需要审批的单据，点击"全部驳回"
- **并且** 填写备注信息后点击"确定"
- **那么** 页面应提示"操作成功"
- **并且** 支持批量选择多条单据后进行批量驳回
- **并且** 驳回后该单据从待审批列表中移除
