# 功能: 访问日志 - 流水日志查看

**模块**: 访问日志
**页面**: /gateways/:id/logs
**优先级**: P1
**前置条件**: 已登录，且网关存在访问日志数据

## 场景: 查看访问日志

- **假设** 用户已登录并进入流水日志页面
- **当** 用户点击起止时间框，分别选择开始时间和结束时间
- **并且** 点击"确定"按钮
- **那么** 页面应展示对应时间范围内的流水日志
- **并且** 支持点击日志条目查看详情
- **并且** 支持复制日志链接和日志内容

## 场景: 日志搜索与筛选

- **假设** 用户已进入流水日志页面
- **当** 用户在查询条件搜索框中输入request_id进行搜索
- **那么** 页面应展示匹配的流水日志记录
- **并且** 支持点击"显示示例"查看查询条件示例
- **并且** 支持点击"请求流水查询规则"查看查询语法说明
- **并且** 支持通过环境筛选查看不同环境的日志
