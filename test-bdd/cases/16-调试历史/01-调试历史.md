# 功能: 调试历史 - 请求记录管理

**模块**: 调试历史
**页面**: /gateways/:id/online-debug/history
**优先级**: P2
**前置条件**: 已登录，且存在调试请求历史记录

## 场景: 查看调试历史

- **假设** 用户已登录并进入请求记录页面
- **当** 用户查看调试历史列表
- **那么** 页面应展示历史请求记录列表
- **并且** 支持点击展开图标或"请求详情"查看请求的详细信息
- **并且** 支持点击复制图标复制请求详情内容
- **并且** 支持自定义表格列设置

## 场景: 历史记录筛选

- **假设** 用户已进入请求记录页面
- **当** 用户在搜索框中输入资源名称进行搜索
- **那么** 列表应展示匹配该资源名称的调试历史记录
- **并且** 支持模糊搜索功能
- **并且** 支持通过左下角切换每页显示条数
