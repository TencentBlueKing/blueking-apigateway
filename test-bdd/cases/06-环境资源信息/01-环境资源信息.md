# 功能: 环境资源信息 - 环境资源信息

**模块**: 环境资源信息
**页面**: /gateways/:id/stages/:stage/resources
**优先级**: P2
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在环境且已发布资源

## 场景: 查看环境资源列表

- **假设** 用户已进入环境概览页面，切换到详情模式并选择环境
- **当** 用户查看资源信息列表
- **那么** 页面展示该环境下已发布的资源列表，包含资源名称、前端请求方法、前端请求路径、后端服务等信息

## 场景: 资源筛选

- **假设** 用户已进入环境资源信息页面，已存在多个资源
- **当** 用户在搜索框中输入资源名称或前端请求路径进行搜索
- **那么** 页面展示匹配的资源
- **当** 用户点击前端请求方法旁的筛选框，勾选请求方法（如GET）并确认
- **那么** 页面仅展示前端请求方法为GET的资源
- **当** 用户点击重置
- **那么** 筛选条件清空，展示全部资源

## 场景: 查看资源详情

- **假设** 用户已进入环境资源信息页面
- **当** 用户选择资源查看资源详情
- **那么** 可正常查看资源的详细配置信息
- **并且** 支持复制资源地址
