# 功能: 全站路由冒烟 - 无 500 / 无失败页

**模块**: 全站路由冒烟
**页面**: /, /:id/*, /components/*, /platform-tools/*, /docs/*, /mcp-market, /personal-workbench/*
**优先级**: P0
**类型**: Read-only

## 前置条件

- BDD 全局 setup 已创建测试网关、默认环境、后端服务、测试资源并发布到 prod
- 使用当前前端路由清单作为页面覆盖基准

## 场景: 访问所有静态页面和可解析动态页面

**假设** 测试用户已登录
**并且** 测试网关已完成基础数据初始化
**当** 依次访问网关管理、资源、环境、权限、运行数据、监控告警、MCP、组件管理、API 文档、平台工具、MCP 市场、个人工作台的所有静态路由
**并且** 对资源详情、MCP 详情、组件运行数据详情、MCP 市场详情等动态路由使用当前环境可解析的数据访问
**那么** 每个页面都应正常加载主内容
**并且** 页面不应出现 500、502、503、504、404、Server Error、系统异常或失败页
**并且** 页面加载期间不应出现致命前端 pageerror
**并且** 页面加载期间不应出现 dashboard 后端接口 5xx 响应
