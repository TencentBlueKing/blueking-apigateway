# 功能: SDK列表 - SDK列表

**模块**: SDK列表
**页面**: /gateways/:id/resource-versions/sdks
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关和资源版本

## 场景: 查看SDK列表

- **假设** 用户已进入SDK列表页面
- **当** 用户点击生成SDK，选择资源版本，输入SDK版本号，选择语言（如Python）
- **并且** 点击确定
- **那么** SDK创建成功，SDK列表中展示新生成的SDK
- **并且** 支持复制SDK地址和下载SDK

## 场景: SDK筛选

- **假设** 用户已进入SDK列表页面，已生成多个SDK
- **当** 用户通过SDK版本号进行搜索
- **那么** 页面展示匹配SDK版本号的记录
- **当** 用户通过资源版本号进行搜索
- **那么** 页面展示对应资源版本的SDK记录
- **当** 用户输入不存在的版本号搜索
- **那么** 页面展示空结果
