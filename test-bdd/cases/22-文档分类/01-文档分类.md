# 功能: 文档分类 - 文档分类管理

**模块**: 文档分类
**页面**: /docs/categories
**优先级**: P1
**前置条件**: 已登录

## 场景: 编辑文档分类

- **假设** 用户已登录并进入文档分类页面
- **当** 用户选择文档点击"编辑"
- **并且** 编辑优先级（范围1-9999）后保存
- **那么** 页面应提示"编辑成功"
- **并且** 优先级不能为空，超出范围应提示错误
- **并且** 支持自定义表格列设置

## 场景: 管理分类

- **假设** 用户已进入文档分类页面
- **当** 用户在搜索框中输入文档分类名称并按Enter搜索
- **那么** 列表应展示匹配的文档分类记录
- **并且** 支持模糊搜索功能
- **并且** 支持通过左下角切换每页显示条数
