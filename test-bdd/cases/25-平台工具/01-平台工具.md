# 功能: 平台工具 - 平台工具集

**模块**: 平台工具
**页面**: /platform-tools/toolbox, /platform-tools/automated-gateway, /platform-tools/bk-cli, /platform-tools/programmable-gateway, /platform-tools/micro-gateway
**优先级**: P1
**前置条件**: 已登录

## 场景: 工具箱本地转换工具

- **假设** 用户已登录并进入工具箱页面
- **当** 用户使用 Base64、URL、JSON、JWT 工具
- **那么** 页面展示对应转换结果

## 场景: 工具箱日志和 Trace 查询入口

- **假设** 用户已登录并进入工具箱页面
- **当** 用户输入不存在的 request_id 查询日志和 Trace
- **那么** 页面展示查询表单和错误/空态

## 场景: 查看平台工具子页

- **假设** 用户已登录并进入平台工具页面
- **当** 用户分别打开自动化接入网关、CLI 工具、可编程网关、蓝鲸微网关
- **那么** 页面路由和内容均正常渲染

## 场景: 查看 BK-CLI 文档标签页

- **假设** 用户已登录并进入 BK-CLI 工具页面
- **当** 用户切换"功能概览"、"快速开始"、"进阶用法"标签页
- **那么** 页面应展示对应 BK-CLI 文档内容和命令示例
