# 功能: 平台工具 - 平台工具集

**模块**: 平台工具
**页面**: /tools/toolbox, /tools/auto-access, /tools/programmable
**优先级**: P1
**前置条件**: 已登录

## 场景: 查询日志工具箱

- **假设** 用户已登录并进入工具箱页面
- **当** 用户在查询日志搜索框中输入request_id进行查询
- **那么** 页面应展示匹配的日志数据
- **并且** 支持JWT解析功能，输入JWT内容后展示解析结果
- **并且** 支持JSON格式化功能，输入JSON内容后展示格式化结果

## 场景: 查看自动化接入与可编程网关

- **假设** 用户已登录并进入平台工具页面
- **当** 用户在"自动化接入网关"页面点击"查看详情"
- **那么** 页面应跳转到自动化接入网关的GitHub文档页面
- **并且** 在"可编程网关"页面点击"查看详情"应跳转到可编程网关的GitHub文档页面
