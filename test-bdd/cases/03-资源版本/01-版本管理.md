# 功能: 资源版本 - 版本管理

**模块**: 资源版本
**页面**: /gateways/:id/resource-versions
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关和资源

## 场景: 生成版本

- **假设** 用户已进入资源版本页面，已存在可用资源
- **当** 用户点击生成版本，输入版本号和版本说明
- **并且** 点击确定
- **那么** 版本生成成功，版本列表中展示新生成的版本
- **并且** 可选择将版本发布至指定环境

## 场景: 查看版本列表

- **假设** 用户已进入资源版本页面，已生成多个版本
- **当** 用户在搜索框中输入版本号进行搜索
- **那么** 页面展示匹配的版本记录
- **当** 用户点击版本号
- **那么** 跳转到版本资源详情页面，可查看该版本包含的资源列表

## 场景: 版本对比

- **假设** 用户已进入资源版本页面，已生成两个以上版本
- **当** 用户勾选两个版本，点击版本对比
- **那么** 页面展示两个版本的资源对比信息
- **并且** 支持按资源名称、资源地址或请求路径搜索资源
- **并且** 支持按差异类型筛选，支持勾选仅显示有差异的资源属性

## 场景: 版本详情

- **假设** 用户已进入资源版本页面，已生成版本
- **当** 用户选择版本并点击发布至环境
- **并且** 选择目标环境，点击确认，下一步确认发布
- **那么** 版本发布成功
