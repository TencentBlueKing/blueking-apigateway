# 功能: 环境插件管理 - 环境插件管理

**模块**: 环境插件管理
**页面**: /gateways/:id/stages/:stage/plugins
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在环境

## 场景: 添加环境插件

- **假设** 用户已进入环境概览页面，切换到详情模式并选择环境
- **当** 用户点击添加插件，选择插件类型（如跨域资源共享CORS、频率控制、Header转换、IP访问控制等）
- **并且** 点击下一步，填写插件配置信息
- **并且** 点击确定
- **那么** 插件添加成功，插件列表中展示新添加的插件

## 场景: 编辑环境插件

- **假设** 用户已进入环境插件管理页面，已存在已添加的插件
- **当** 用户鼠标悬浮已有插件，点击编辑（铅笔图标）
- **并且** 修改插件配置信息，点击确定
- **那么** 页面提示修改成功

## 场景: 删除环境插件

- **假设** 用户已进入环境插件管理页面，已存在已添加的插件
- **当** 用户鼠标悬浮已有插件，点击删除（垃圾桶图标）
- **并且** 确认停用
- **那么** 插件停用成功，插件从列表中移除

## 场景: 查看插件列表

- **假设** 用户已进入环境插件管理页面
- **当** 用户点击添加插件，在搜索框中输入插件关键字并按Enter搜索
- **那么** 页面展示匹配的插件（已添加的插件显示为置灰状态）
- **当** 用户切换不同插件类型标签页查看
- **那么** 页面展示对应类型的插件列表
