# 功能: MCP服务 - MCP Server管理

**模块**: MCP服务
**页面**: /gateways/:id/mcp
**优先级**: P1
**前置条件**: 已登录，且存在已发布的资源

## 场景: 创建MCP Server

- **假设** 用户已登录并进入MCP Server页面
- **当** 用户点击"+ 新建"按钮
- **并且** 输入服务名称、服务展示名、描述，选择分类、连接方式，勾选资源和Prompt后点击"确定"
- **那么** 页面应提示"创建成功"
- **并且** 服务名称只能包含小写字母、数字和短横线，否则提示参数错误
- **并且** 服务名称长度限制为3-32个字符，描述长度限制为10-2048个字符

## 场景: 编辑MCP Server

- **假设** MCP Server列表中存在已创建的MCP Server
- **当** 用户选择MCP Server点击"编辑"
- **并且** 修改描述、标签、资源等信息后点击"确定"
- **那么** 页面应提示"编辑成功"
- **并且** 支持点击"取消"放弃编辑

## 场景: 删除MCP Server

- **假设** MCP Server已停用
- **当** 用户选择该MCP Server点击"删除"
- **并且** 点击"确定"确认删除
- **那么** 该MCP Server应被成功删除
- **并且** 启用状态的MCP Server需先停用后才能删除

## 场景: 查看MCP列表

- **假设** 用户已进入MCP Server页面
- **当** 用户查看MCP Server列表
- **那么** 页面应展示所有MCP Server及其状态信息
- **并且** 支持通过名称模糊搜索MCP Server
- **并且** 支持通过环境、分类、标签进行筛选
- **并且** 支持按发布时间排序
- **并且** 支持停用和启用MCP Server
- **并且** 支持复制MCP Server地址
- **并且** 支持自定义表格列设置
