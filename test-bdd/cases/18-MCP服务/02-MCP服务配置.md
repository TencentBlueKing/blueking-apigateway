# 功能: MCP服务 - MCP Server配置

**模块**: MCP服务
**页面**: /gateways/:id/mcp/detail/:serverId
**优先级**: P2
**前置条件**: 已登录，且已创建MCP Server

## 场景: 查看详情、Tool、接入配置和自定义文档

- **假设** 用户已进入MCP Server详情页
- **当** 用户查看详情页、Tool 列表、接入配置和使用指引
- **并且** 新增、编辑、删除用户自定义文档
- **那么** Tool 列表、接入配置、使用指引、自定义文档接口返回与操作一致
- **并且** Prompt 只在远程 Prompt 接口可用时观察，不强依赖外部服务
