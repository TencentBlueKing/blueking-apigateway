# 功能: MCP服务 - MCP Server配置

**模块**: MCP服务
**页面**: /gateways/:id/mcp
**优先级**: P2
**前置条件**: 已登录，且已创建MCP Server

## 场景: 配置连接方式

- **假设** 用户已进入MCP Server详情页
- **当** 用户查看使用指引中的配置信息
- **那么** 页面应展示各客户端（如Cursor等）的配置信息
- **并且** 支持复制各项配置信息
- **并且** 支持查看权限审批相关配置
- **并且** 支持管理已授权应用的权限（搜索、删除应用权限）

## 场景: 配置资源与Prompt

- **假设** 用户正在创建或编辑MCP Server
- **当** 用户在资源选择区域勾选需要的资源
- **那么** 已勾选的资源应显示在已选列表中
- **并且** 支持刷新资源列表和分页浏览资源
- **并且** 支持点击"清空"按钮清除已选资源
- **并且** 支持在Prompt区域通过中英文名、标签、内容搜索Prompt
- **并且** 支持修改Tool的名称信息
- **并且** 支持在Tool列表中搜索工具和查看文档详情
