# 功能: 基本信息 - 网关基本信息管理

**模块**: 基本信息
**页面**: /gateways/:id/basic
**优先级**: P1
**前置条件**: 已登录，且已进入网关基本信息页面

## 场景: 查看基本信息

- **假设** 用户已登录并进入基本信息页面
- **当** 用户查看网关基本信息
- **那么** 页面应展示网关名称、描述、维护人员、是否公开等基础信息
- **并且** 支持复制网关API地址
- **并且** 支持复制文档地址和点击文档地址跳转
- **并且** 支持点击"更多详情"跳转到X-Bkapi-JWT概念说明
- **并且** 支持复制和下载API公钥

## 场景: 编辑基本信息

- **假设** 用户已进入基本信息页面
- **当** 用户点击"编辑基础信息"
- **并且** 修改维护人员、描述、是否公开等信息后点击"保存"
- **那么** 页面应提示"编辑成功"
- **并且** 支持修改API文档相关设置
- **并且** 支持点击"取消"放弃编辑

## 场景: 网关状态管理

- **假设** 用户已进入基本信息页面
- **当** 用户点击"停用"按钮并确认
- **那么** 网关应停用成功
- **并且** 停用后支持点击"立即启用"按钮重新启用网关
- **并且** 网关停用状态下支持点击"删除"按钮，输入网关名称确认后删除网关
