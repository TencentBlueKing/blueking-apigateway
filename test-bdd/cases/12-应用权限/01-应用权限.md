# 功能: 应用权限 - 应用权限管理

**模块**: 应用权限
**页面**: /gateways/:id/permission/app
**优先级**: P1
**前置条件**: 已登录，测试会创建唯一应用权限数据

## 场景: 主动授权、续期和删除应用权限

- **假设** 用户已登录并进入应用权限页面
- **当** 用户对唯一 bk_app_code 主动授权网关权限和资源权限
- **并且** 对单个网关权限续期
- **并且** 对网关权限和资源权限批量续期
- **并且** 删除该应用的网关权限和资源权限
- **那么** API 权限列表中应能确认授权、续期和删除结果

## 场景: 权限筛选

- **假设** 用户已进入应用权限页面
- **当** 用户查看应用权限列表和空态
- **那么** 应用权限接口返回成功
- **并且** 页面展示表格或空态
