# 功能: 应用权限 - 应用权限管理

**模块**: 应用权限
**页面**: /gateways/:id/permissions/apps
**优先级**: P1
**前置条件**: 已登录，且存在已授权的应用权限记录

## 场景: 查看应用权限列表

- **假设** 用户已登录并进入应用权限页面
- **当** 用户点击"权限管理"-"应用权限"菜单
- **那么** 页面应展示已授权的应用权限列表
- **并且** 列表中应显示蓝鲸应用ID、资源名称、授权维度、过期时间等信息
- **并且** 支持通过左下角切换每页显示条数
- **并且** 支持点击"导出全部"导出所有应用权限数据

## 场景: 管理应用权限

- **假设** 应用权限列表中存在已授权的权限记录
- **当** 用户勾选应用权限记录，点击"批量续期"
- **并且** 选择续期时长（永久/6个月/12个月）后点击"确定"
- **那么** 页面应提示"续期成功"
- **并且** 支持单个应用权限的续期操作
- **并且** 支持选择单个应用权限点击"删除"并确认后删除成功

## 场景: 权限筛选

- **假设** 用户已进入应用权限页面
- **当** 用户点击搜索框，选择蓝鲸应用ID或资源名称进行搜索
- **那么** 列表应展示匹配的应用权限记录
- **并且** 支持通过授权维度筛选框选择"按网关"或"按资源"进行筛选
- **并且** 支持模糊搜索功能
- **并且** 支持自定义表格列设置
