# 功能: MCP权限审批 - MCP权限审批管理

**模块**: MCP权限审批
**页面**: /gateways/:id/mcp/approvals
**优先级**: P2
**前置条件**: 已登录，且存在MCP Server，且开发者中心已发起MCP权限申请

## 场景: MCP权限审批通过

- **假设** MCP权限审批列表中存在待审批的权限申请
- **当** 用户点击"权限审批"，选择待审批单据点击"通过"
- **并且** 点击"确定"
- **那么** 页面应提示"操作成功"
- **并且** 支持通过蓝鲸应用ID搜索待审批单据
- **并且** 支持通过MCP Server名称筛选审批单据
- **并且** 支持分页浏览审批列表

## 场景: MCP权限审批驳回

- **假设** MCP权限审批列表中存在待审批的权限申请
- **当** 用户选择待审批单据点击"通过"后点击"取消"
- **那么** 审批操作应被取消，单据保持待审批状态
- **并且** 支持通过蓝鲸应用ID和MCP Server名称组合筛选审批单据
