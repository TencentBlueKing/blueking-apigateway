# 功能: 环境概览 - 环境概览

**模块**: 环境概览
**页面**: /gateways/:id/stages
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在网关

## 场景: 查看环境概览

- **假设** 用户已进入环境概览页面
- **当** 用户点击"+"新建环境
- **并且** 输入环境名称（必填）、描述（选填），选择负载均衡类型，输入后端服务地址和超时时间
- **并且** 点击确定
- **那么** 环境新建成功，环境概览中展示新建的环境

## 场景: 发布资源

- **假设** 用户已进入环境概览页面，已存在环境且已有资源版本
- **当** 用户选择环境并点击发布资源
- **并且** 选择要发布的资源版本，点击下一步
- **并且** 确认发布
- **那么** 资源发布成功

## 场景: 下架资源

- **假设** 用户已进入环境概览页面，环境已发布资源
- **当** 用户选择环境并点击下架
- **并且** 确认下架操作
- **那么** 资源下架成功，环境恢复为未发布状态
