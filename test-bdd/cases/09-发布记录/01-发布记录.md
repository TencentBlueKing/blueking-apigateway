# 功能: 发布记录 - 发布记录

**模块**: 发布记录
**页面**: /gateways/:id/releases
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在发布操作记录

## 场景: 查看发布记录

- **假设** 用户已进入发布记录页面，已存在发布操作记录
- **当** 用户查看发布记录列表
- **那么** 页面展示所有发布记录，包含环境、版本、发布时间等信息
- **当** 用户点击某条发布记录的发布日志
- **那么** 可正常查看发布日志详情

## 场景: 记录筛选

- **假设** 用户已进入发布记录页面，已存在多条发布记录
- **当** 用户输入环境名称（如prod）进行搜索
- **那么** 页面展示该环境下的发布记录
- **当** 用户点击时间选择框，选择开始时间和结束时间并确定
- **那么** 页面展示对应时间范围内的发布记录
- **当** 用户点击清除按钮
- **那么** 时间筛选条件清空，展示全部发布记录
