# 功能: 网关管理 - 网关列表

**模块**: 网关管理
**页面**: /
**优先级**: P2
**前置条件**: 用户已登录蓝鲸API网关管理端，已存在多个网关

## 场景: 搜索网关

- **假设** 用户已登录并进入我的网关页面，已存在多个网关
- **当** 用户在搜索框中输入网关名称关键字进行搜索
- **那么** 页面展示所有包含该关键字的网关（支持模糊搜索）
- **当** 用户输入不存在的网关名称进行搜索
- **那么** 页面展示空结果

## 场景: 查看网关详情

- **假设** 用户已登录并进入我的网关页面，已存在至少一个网关
- **当** 用户点击对应网关的资源数量
- **那么** 跳转到资源管理-资源配置页面，实际资源数量与网关中显示的资源数量一致
- **当** 用户点击对应网关的环境概览
- **那么** 跳转到对应网关环境管理-环境概览页面
- **当** 用户通过网关类型筛选（如选择普通网关或可编程网关）
- **那么** 页面仅展示对应类型的网关
