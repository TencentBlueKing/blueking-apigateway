# 功能: 网关管理 - 创建网关

**模块**: 网关管理
**页面**: /
**优先级**: P1
**前置条件**: 用户已登录蓝鲸API网关管理端

## 场景: 创建普通网关

- **假设** 用户已登录并进入我的网关页面
- **当** 用户点击新建网关，网关类型选择普通网关
- **并且** 输入网关名称、维护人员，选填描述，选择公开
- **并且** 点击确定
- **那么** 网关创建成功，用户可查看资源文档、申请资源权限

## 场景: 创建可编程网关

- **假设** 用户已登录并进入我的网关页面
- **当** 用户点击新建网关，网关类型选择可编程网关
- **并且** 输入网关名称、维护人员，选填描述
- **并且** 选择开发语言，输入可用的代码仓库地址、账号、密码
- **并且** 选择公开，点击确定
- **那么** 网关创建成功，用户可查看资源文档、申请资源权限

## 场景: 创建私有网关

- **假设** 用户已登录并进入我的网关页面
- **当** 用户点击新建网关，网关类型选择普通网关
- **并且** 输入网关名称、维护人员，是否公开选择不公开（开关按钮置灰）
- **并且** 点击确定
- **那么** 网关创建成功且网关对用户隐藏，用户无法在公开列表中查看该网关

## 场景: 名称校验

- **假设** 用户已登录并进入新建网关页面
- **当** 用户未填写网关名称并尝试提交
- **那么** 页面提示"请填写名称"
- **当** 用户输入的网关名称首字符不为小写字母且字符长度大于3
- **那么** 页面提示"由小写字母、数字、连接符（-）组成，首字符必须是小写字母，长度大于3小于30个字符"
