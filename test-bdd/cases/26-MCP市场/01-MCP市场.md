# 功能: MCP市场 - MCP Server市场浏览

**模块**: MCP市场
**页面**: /mcp-market
**优先级**: P2
**前置条件**: 已登录，且存在已发布的MCP Server

## 场景: 浏览MCP市场

- **假设** 用户已登录并进入MCP市场页面
- **当** 用户在搜索框中输入MCP Server名称或描述进行搜索
- **那么** 页面应展示匹配的MCP Server列表
- **并且** 支持点击MCP Server进入详情页查看工具列表和Prompt
- **并且** 支持在详情页中搜索工具名称和描述
- **并且** 支持查看工具的文档详情
- **并且** 支持复制配置信息（如Cursor等客户端配置）
- **并且** 支持查看权限申请指引
- **并且** 支持按更新时间排序MCP Server

## 场景: MCP分类筛选

- **假设** 用户已进入MCP市场页面
- **当** 用户选择MCP Server分类
- **那么** 页面应展示对应分类下的MCP Server列表
- **并且** 不存在该分类的MCP Server时应显示空状态
