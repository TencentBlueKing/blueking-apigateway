# 功能: API文档 - API文档查看

**模块**: API文档
**页面**: /api-docs/gateways, /api-docs/components
**优先级**: P2
**前置条件**: 已登录

## 场景: 搜索网关API文档

- **假设** 用户已登录并进入网关API文档页面
- **当** 用户在搜索框中输入网关名称或描述进行搜索
- **那么** 页面应展示匹配的网关API列表（支持模糊搜索）
- **并且** 支持点击网关查看已发布到环境的API文档
- **并且** 支持在文档详情中通过资源名称搜索资源
- **并且** 支持自定义表格列设置

## 场景: 查看组件API文档

- **假设** 用户已登录并进入组件API文档页面
- **当** 用户选择组件（如监控平台V3）
- **那么** 页面应正常展示该组件的API文档
- **并且** 支持通过API名称搜索API文档
- **并且** 支持在API列表中搜索和筛选

## 场景: 查看SDK文档

- **假设** SDK已生成
- **当** 用户选择网关或组件点击"查看SDK"
- **那么** 页面应正常展示SDK信息
- **并且** 支持复制SDK地址
- **并且** 支持下载SDK
- **并且** 支持复制访问地址
