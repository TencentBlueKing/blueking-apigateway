### 描述

获取网关最新的资源版本

### 输入参数

#### 路径参数

| 参数名称         | 参数类型 | 必选 | 描述   |
|--------------| -------- | ---- | ------ |
| gateway_name | string   | 是   | 网关名 |


### 响应示例

```json
{
    "data": {
        "version": "1.0.1"
    }
}
```

### 响应参数说明

| 字段    | 类型   | 描述                               |
| ------- | ------ | ---------------------------------- |
| data    | object | 结果数据，详细信息请见下面说明     |

#### data

| 名称    | 类型   | 说明             |
| ------- | ------ | ---------------- |
| version | string | 资源版本的版本号 |
