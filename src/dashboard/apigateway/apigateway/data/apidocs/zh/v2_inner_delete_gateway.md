### 描述

删除网关，仅支持 bp- 开头的网关

> 注意：
> - 此接口仅允许操作以 `bp-` 开头的网关，避免误操作其他网关。
> - 网关必须处于停用状态才能删除，请先调用 `v2_inner_update_gateway_status` 接口停用网关。
> - 删除操作会删除网关及其关联的所有数据（包括资源、环境、版本等），请谨慎操作。


### 输入参数

#### 路径参数

| 参数名称         | 参数类型   | 必选 | 描述   |
|--------------|--------|----|------|
| gateway_name | string | 是  | 网关名称，必须以 bp- 开头 |


### 响应示例

```json
{}
```

成功时返回 HTTP 204 状态码，响应体为空。


### 响应参数说明

| 字段 | 类型 | 描述 |
|------|------|------|
| -    | -    | 成功时返回 HTTP 204，无响应体 |


### 错误码

| 错误码 | 描述 |
|--------|------|
| 400    | 网关名称不以 bp- 开头，或网关未停用 |
| 404    | 网关不存在 |
