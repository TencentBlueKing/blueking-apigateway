### 描述

更新网关状态（停用/启用），仅支持 bp- 开头的网关

> 注意：此接口仅允许操作以 `bp-` 开头的网关，避免误操作其他网关。停用网关时，会自动将网关下所有 MCPServer 设置为停用状态。


### 输入参数

#### 路径参数

| 参数名称         | 参数类型   | 必选 | 描述   |
|--------------|--------|----|------|
| gateway_name | string | 是  | 网关名称，必须以 bp- 开头 |

#### 请求体参数

| 参数名称 | 参数类型 | 必选 | 描述                           |
|--------|--------|----|------------------------------|
| status | int    | 是  | 网关状态，0：停用，1：启用 |


### 请求示例

```json
{
    "status": 0
}
```


### 响应示例

```json
{}
```

成功时返回 HTTP 204 状态码，响应体为空。


### 响应参数说明

| 字段 | 类型 | 描述 |
|------|------|------|
| -    | -    | 成功时返回 HTTP 204，无响应体 |


### 错误码

| 错误码 | 描述 |
|--------|------|
| 400    | 网关名称不以 bp- 开头，或参数校验失败 |
| 404    | 网关不存在 |
