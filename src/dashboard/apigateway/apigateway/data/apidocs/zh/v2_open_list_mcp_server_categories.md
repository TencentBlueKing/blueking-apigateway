### 描述

获取所有已激活的 MCPServer 分类列表

供前端展示分类筛选下拉框，或用于其他系统获取可用分类信息。

### 输入参数

无

### 响应示例

```json
{
    "data": [
        {
            "name": "official",
            "display_name": "官方"
        },
        {
            "name": "community",
            "display_name": "社区"
        }
    ]
}
```

### 响应参数说明

| 字段 | 类型  | 描述                           |
| ---- | ----- | ------------------------------ |
| data | array | 结果数据，详细信息请见下面说明 |

#### data[]

| 参数名称     | 参数类型 | 描述                   |
| ------------ | -------- | ---------------------- |
| name         | string   | 分类名称（英文标识）   |
| display_name | string   | 分类显示名称           |
