## Installing

### maven
```
<dependency>
    <groupId>com.tencent.bkapi</groupId>
    <artifactId>{gateway_name}</artifactId>
    <version>{sdk_version}</version>
</dependency>
```

### grade

```
implementation 'com.tencent.bkapi:{gateway_name}:{sdk_version}'

```
