### 响应示例

```json
{
    "code": 0,
    "message": "OK",
    "data": [
        {
            "id": 1,
            "name": "bk-apigateway",
            "description": "",
            "maintainers": [
                "admin"
            ]
        }
    ]
}
```

### 响应参数说明

| 字段    | 类型   | 描述                               |
| ------- | ------ | ---------------------------------- |
| code    | int    | 返回码，0 表示成功，其它值表示失败 |
| message | string | 错误信息                           |
| data    | array  | 结果数据，详细信息请见下面说明     |

#### data

| 参数名称    | 参数类型 | 描述       |
| ----------- | -------- | ---------- |
| id          | int      | 网关ID     |
| name        | string   | 网关名称   |
| description | string   | 网关描述   |
| maintainers | array    | 网关管理员 |
